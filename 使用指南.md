# 数学建模助教 · 使用指南

写给**使用者**。一句话：下载 → 双击 → 跟着向导连一次大模型 → 开始用。无需懂编程，无需装 Python。

---

## 一、下载

挑你电脑的系统，下对应的包（从 GitHub Releases 或维护者给的云盘链接）：

| 你的系统 | 下载哪个 |
|---------|---------|
| **Windows** | `数学建模助教-安装程序-vX.X.X.exe`（推荐，图形安装）<br>或 `数学建模助教-windows-x64.zip`（免安装，解压即用） |
| **macOS** | `数学建模助教-macos-*.tar.gz` |
| **Linux** | `数学建模助教-linux-*.tar.gz` |

> 包比较大（约几百 MB），因为它**自带了完整运行环境**，你不用再单独装任何东西。

---

## 二、安装 / 启动

### Windows（安装程序）
1. 双击 `数学建模助教-安装程序-vX.X.X.exe`。
2. 一路「下一步」（默认装在当前用户目录，不需要管理员权限）。
3. 安装完从**桌面**或**开始菜单**点「数学建模助教」启动。

### Windows（免安装 zip）
1. 把 zip 解压到任意目录（如桌面）。
2. 进入解压出的文件夹，双击 **`启动助教.exe`**（或 `双击我启动.bat`）。

### macOS / Linux
1. 解压 `.tar.gz`。
2. 终端进入目录，运行 `./启动助教`。
3. macOS 若提示「无法打开」：到「系统设置 › 隐私与安全性」点「仍要打开」，或在该目录执行
   `xattr -dr com.apple.quarantine .` 后重试。

启动后会弹出一个黑色窗口（这是程序在运行，别关），等十几秒**浏览器会自动打开**。
没自动打开就手动访问窗口里显示的地址（形如 `http://127.0.0.1:8000/`）。

## 三、第一次使用：连接 AI 大模型（只需一次）

聊天、跑代码都要连一个大模型。第一次进主页会弹出**「连接 AI 大模型」向导**，两种方式**任选一种**：

### 方式 ①：用我们提供的额度（最省事）
1. 点「用我们提供的额度」。
2. 点「打开注册站点领取额度」，在站点注册账号、领/兑换额度。
3. 在站点生成一个 API Key（形如 `sk-…`），复制。
4. 回到向导，粘贴 Key，点「保存并开始」。接口地址和模型会自动填好。

> 如果这个按钮显示「未开启」，说明维护者还没提供额度服务，请用方式 ②。

### 方式 ②：用我自己的 API Key
1. 点「用我自己的 API Key」。
2. 选服务商（DeepSeek / 通义千问 / OpenAI / Moonshot / 自定义）。
   - 点「去官网申请 API Key」可直接跳到对应平台申请页。
3. 把你的 Key 粘进去（接口地址、模型已按服务商预填好）。
4. 点「保存并开始」。

> 想以后更换：主页右上角「API 设置」随时可以重新配置。

配好之后，向导关闭，正常进入产品功能引导，就能开始用了。

## 四、四个功能

- **学习模式**：顺着知识图谱，从基础概念一步步点亮。
- **做建模**：自由练习工作台（编辑器 + 终端 + AI 对话），或带一道题让助教拆步、陪你一步步做。
- **真题练习**：历年真题分步练，提交后 AI 按评分要点给分点评。
- **AI 对话**：随时提问，支持公式渲染、Python 代码自动运行、上传 PDF/Word/Excel 等文件。

## 五、你的数据存在哪 / 怎么退出

- **退出**：关掉那个黑色启动窗口即可。
- **数据位置**（聊天记录、档案、做题进度，卸载程序**不会**删除）：
  - Windows：`%LOCALAPPDATA%\MathModelingAgent\data`
  - macOS：`~/Library/Application Support/MathModelingAgent/data`
  - Linux：`~/.local/share/MathModelingAgent/data`

## 六、常见问题

**Q：杀毒软件/SmartScreen 报警？**
因为启动器没有代码签名证书。点「更多信息 › 仍要运行」即可；介意的话用免安装 zip 版。

**Q：浏览器没自动打开？**
手动访问启动窗口里显示的地址（`http://127.0.0.1:8000/`，端口可能不同）。

**Q：提示端口被占用？**
程序会自动换一个空闲端口，以窗口里显示的地址为准。

**Q：能离线用吗？**
知识库、跑代码是本地的；但聊天必须联网连大模型。

**Q：跑代码安全吗？**
模型生成的代码在你本机以你的权限运行，运行前界面会展示要执行的代码，请只在信任环境使用。
